AVE FLV MediaPlayer v1.1
===========================

Что нового:

1.Для управления диалоговым меню плеера подключена библиотека JQUERY. В дальнейшем это позволит существенно расширить функциональность окна.
Внимание: путь к файлу jquery.js, при необходимости вы сможете отредактировать в файле admin/editor/editor/plugins/flvS3Player/flvS3Player.html 
2.Добавились два новых поля :
2.1. Комментарий к файлу (исполнитель, имя файла и т.п.), вывод в левом верхнем углу плеера. 
Поле необязательное , можно оставлять его пустым. Можно просто использовать текст, но можно и HTML код, что позволяет вам "встраивать" даже ссылки.
Можно использовать теги:
<b> </b>— полужирный
    <i> </i>— наклонный
    <u> </u>— подчеркивание
    <br> — перевод строки
    <font> </font> — форматирование с параметрами size (размер текста), color (цвет текста), face (шрифт).
    <a> </a> — ссылка с параметрами href (сама ссылка) и target (в каком окне открывать _self в своем, _blank в новом)


Например, в диалоговом окне плеера, в поле "Комментарий к файлу" можно вставить вот такой вот код:

<font size='18px'>В названии</font> можно использовать <font face='Georgia' size='16px' color='#ff0000'><b><i>HTML</i></b></font> и <a href='http://overdoze.ru' target='_blank'><u>ссылки</u></a>

*Для тестирования данного примера рекомендую использовать файл стиля video505x380_5.txt

2.2. Добавлена возможность загружать плей-листы , причем как внутренние , так и с внешних серверов.)
ВАЖНО: Плейлист пишется и хранится в текстовом файле в кодировке UTF-8 (иначе получите "крокозябры!) поэтому при создании и сохранении плейлистов используйте соответствующий редактор, позволяющий выбирать кодировку сохраняемого файла.
Структура плейлиста:
1.Плейлист - это текстовая строка в формате JSON, которая представляет собой список (массив) параметров каждого файла. 

Формат:

{"playlist":[{ },{ },…{ }]}

Опции каждого файла перечисляются в фигурных скобках { } через запятую в формате «параметр»:"значение". Например:
 
{"playlist":[{"comment":"Название1","file":"file1.flv","poster":"img1.jpg"},{"comment":"Название2","file":"file2.flv","poster":"img2.jpg"}]}

Параметры, которые можно использовать:

    comment — комментарии к файлу;
    file — ссылка на файл (используйте прямые ссылки через http, если плеер можно расшаривать);
    poster — картинка для плейлиста (JPG, GIF, PNG);
    bigposter — картинка для заставки (если используется). 

Пример готового плейлиста положил в папку uploads/video/playlist/playlist.txt
