Plugin for FSKEditor ----- Insert Icon v1.0 -----

Плагин позволяет вставлять иконки из диалогового окна в редактор FSKEditor.

                Инструкция для AVE CMS v2.09 (да и собственно вообще по работе плагина)

1. Подготовка заключается только в том, что Вам необходимо распаковать архивы с иконками в следующих папках:

1.1. uploads/insert_icons/preview/preview.rar
1.2. uploads/insert_icons/size_16x16/size_16x16.rar
1.2. uploads/insert_icons/size_24x24/size_24x24.rar
1.2. uploads/insert_icons/size_32x32/size_32x32.rar
1.2. uploads/insert_icons/size_48x48/size_48x48.rar

После распаковки всех этих архивов, плагин готов к работе.

               Краткое описание работы.

1. При нажатии на иконку плагина в тулбаре редактора, откроется диалоговое окно, где будут отображены все доступные иконки. Копирование выбранной иконки в поле редактора осуществляется простым кликом по нужной иконке. Вверху окна расположены четыре чекбокса для выбора размеров иконок. По умолчанию активен чекбокс соответствующий размеру изображения иконки 48х48 пикселей. Для того, что-бы изменить размер - кликните по нужному чекбоксу -  сделав его активным тем самым, далее кликните по нужной иконке. В редактор будет вставлена иконка размером, выбранном Вами.

2. Папки с иконками. Все папки содержат ОДИНАКОВОЕ количество иконок с ОДИНАКОВЫМИ именами файлов, разница заключается только в том, что размеры самих изображений должны соответствовать названиям папок: например в папке size_32x32/ должны лежать файлы с изображениями (иконками) 32х32 пикселя, в папке size_24x24/ точно такие же файлы, но с размерами изображений уже 24х24 пикселя, с остальными папками по аналогии. 
2.1. Папка preview/ Ее предназначение отображать в диалоговом окне все доступные иконки и вставлять их в редактор. В эту папку копируются иконки размерами 32х32 пикселя, т.е. файлы в папке  uploads/insert_icons/preview/  и   uploads/insert_icons/size_32x32/ абсолютно одинаковые. Возможно в следующей версии код перепишу и буду вытаскивать файлы из одной папки и для просмотра в диалоговом окне и для вставки в редактор иконок размером 32х32.

               Удаление старых иконок - добавление новых. 
  
1. Если Вы удаляете иконку, например example.png, то все файлы с таким именем должны быть удалены из всех папок, смертельного ничего не произойдет, если вы в какой то папке ее оставите,  но она будет храниться в ней как хлам и занимать место, потому что , если ее не будет в папке preview, вы не увидите эту иконку в диалоговом окне, если она там останется, но будет удалена например в папке size_16x16, тогда кликнув по ней в диалоговом окне в редакторе соответсвенно получите код с отсутствующим файлом.   Добавление новых иконок аналогично удалению....
1.2. После того, как будут удалены или добавлены новые иконки , откройте файл плагина находящийся по следующему пути:  admin/editor/editor/plugins/inserticon/fckplugin.js  Что в нем? В нем находится массив с именами файлов иконок - смотрите строку 
FCKConfig.InsertIconImages  = [ '0','1',........,'zzzyen_currency_sign','zzzzoom_in','zzzzoom_out' ]; 

В этом массиве хранятся данные о всех ваших иконках, при добавлении нового файла добавьте в массив запись 'имя нового файла',  т.е. вы добавляете только имена файлов. Расширение (.png) не пишется! Файлы перечисляются в одинарных кавычках через запятую, после последнего в массиве файла запятая НЕ ставится.

Это пожалуй все.

P.S. На иконки, находящиеся в архивах, распространяется действие лицензий, с текстом которых, Вы можете ознакомиться распаковав архивы. Используйте лицензионные иконки!)
