Версия 2.0

Изменения и дополнения.

Редактор и управление встроены в систему. 
1. На странице "Управление общими настройками системы" выберете Редактор по умолчанию - LiveEditor, сохраните изменения в настройках системы.
2. Перейдите в Настройки LiveEditor (ссылка расположена в левом сайдбаре).
3. При первом заходе на страницу  "Настройки редактора LiveEditor", система автоматически создаст файл ознакомительной (триальной)лицензии, которая
позволяет без каких либо ограничений использовать редактор в течении 30 календарных дней.
4. На открывшейся странице , Вам будут доступны настройки для поля многострочное и многострочное упрощенное.
Видеоуроки вынесены за пределы дистрибутива сборки и загружаются для просмотра с офсайта разработчиков управления редактором.
Вам доступны: настройка панели инструментов (toolbar), просмотр в реальном времени создаваемой панели, + добавлено управление системными 
настройками редактора:
* Вставка <DIV> , <P> или <BR> при нажатии на клавишу Enter, либо поведение по умолчаниям браузера.
* Очистка текста, вставленного из MS WORD.
* Загружать изображения с сервиса Flickr.com либо не использовать эту функцию.
* Начальная загрузка изображений с сервиса Flickr.com - при работе с изображениями, теперь Вы можете указать имя пользователя на  Flickr.com,
если Вы им являетесь , либо Вам требуется загрузка изображений конкретного пользователя, имя которого Вы установите.
* Использовать CSS кнопки ( меню диалога - Ссылки ). Если Вы не хотите использовать ссылки в виде кнопок - Вы можете отключить эту функцию.
* Использовать Автоформат таблицы ( меню диалога - Таблицы ). Эта функция применяет к созданной таблицы готовые стили.  Вы можете отключить эту функцию либо использовать.
* Фильтр CSS правил - выбор селекторов по заданному ПРЕФИКСУ. Новые возможности по фильтрации файла со стилями. Помощь по использованию этой функции
расположена на странице настроек редактора.
* Всегда устанавливать курсор в текстовое поле редактора при загрузке страницы. Если Вы не хотите, что-бы редактор, при открытие страницы сразу получал фокус (устанавливал курсор)в текстовом поле - доступно отключение этой функции.
* Bootstrap - встроен , Вам остается только выбрать из готового)

<link rel="stylesheet" type="text/css" href="[tag:path]lib/bootstrap/css/bootstrap-responsive.min.css" />
