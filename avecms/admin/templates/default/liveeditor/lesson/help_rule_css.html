Вы можете самостоятельно назначать префиксы - другими словами - правила фильтра

Фильтр применяется для файла default.css Путь к файлу: /admin/liveeditor/styles/default.css  Ниже пример использования фильтра:

Если префикс не задан - при нажатии кнопки будет выводиться вся таблица стилей.

Если префикс задан - при нажатии кнопки будет выводиться только та часть таблицы, которая будет соответствовать заданному префиксу.

Например, если будет задан префикс , то будут выбраны ТОЛЬКО селекторы типа:

  • .like_Table {margin-bottom:5px; padding-left:10px;}
  • .like_Div {float:right; padding:6px 0px 10px 8px; width:888px; height:100px;}
  • .like_mod_le_drag_ul { float:left; padding:3px;}

На практике удобно применять фильтр при наличие большого количества стилей в таблице (файл - default.css). Не забывайте, что выбор и задание префикса, подразумевает наличие в таблице стилей селекторов, имя которых соответствует заданным условиям.

В файл /admin/liveeditor/styles/default.css можно добавлять необходимые Вам для работы новые стили.

Внимание: Если Вы не хотите использовать фильтр, оставьте форму для ввода префикса пустой! Любые действия с формой (добавление, удаление, переименование префикса) заканчиваются нажатием кнопки
